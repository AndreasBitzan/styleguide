Naming Conventions

We decided to use BEM as our class naming convention as it is the easiest to read and make edits to. Perfect to be used with Sass.

We used kebab-case for our class names and our file names.

BEM specifies that we do our styling with classes and subclasses that relate to each other. A class in this case would be known as a block, and a subclass would be known as an element of that block. A block is named like you would normally name it, but an element name starts out with the block name then we add a __(element name). For example, "menu-item" would be the block and "menu-item__photo" would edit an element for menu-item called "__photo". 

                
                    .menu-item {
                        display: flex;
                        height: 175px;
                        width: 550px;
                    
                        &__photo {
                            position: relative;
                    
                            height: 175px;
                            width: 175px;
                        }
                    }
