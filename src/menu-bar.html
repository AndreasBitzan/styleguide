The Menu Bar

The menu bar is visible on every screen except for the welcome screen. It features the following rules:

  • The icons are from Fontawesome and must not exceed 50x50px!
  • The space between is evenly which is guaranteed by flexbox!
  • It features a padding of 1rem!
  • Since the project is designed for touch devices, there is no hover effect, but a click effect!
  • The background must be transparent, it is only blue for better visibility here !
  • If there is additional information on the icons, like on how many items are in the shopping cart for example. it must be displayed via a small green bubble containing the information. This can be seen in the example below.
